-  `v1.11.5`版本安装脚本
```
bash <(curl -Ls https://raw.githubusercontent.com/sky22333/xray-singbox/main/singbox/install.sh)
```

> 支持动态传入版本号，例如：`-v 1.11.5`

- 卸载

```
dpkg --purge sing-box
rm -rf /etc/sing-box
```
